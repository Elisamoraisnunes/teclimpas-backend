# TecLimpas - Backend

This is the backend repository for the **Waste Mapping Web Platform (TecLimpas)**, a full-stack application designed to structure geospatial waste data and automate tasks using AI.

## Features
- **AI Integration:** Powered by Gemini LLM APIs for automated waste data processing and categorization.
- **Database & Cloud:** Integrated with Supabase for robust database management and geospatial data structures.
- **RESTful API:** Structured backend utilizing Node.js/Express to serving data efficiently.

## Technologies Used
- **Language:** JavaScript (Node.js)
- **Database:** Supabase / PostgreSQL
- **AI Integration:** Google Gemini API
- **Tools:** Postman, Git/GitHub

## How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/Elisamoraisnunes/teclimpas-backend.git
   
2. Navigate to the project directory:
   ```
   cd teclimpas-backend
   
3. Install dependencies:
   ```
   npm install
   
4. Set up your environment variables (.env):
   ```
   SUPABASE_URL=your_supabase_url
   GEMINI_API_KEY=your_gemini_key
   PORT=3000
   
5.Start the server:
   ```
   npm start
